A digital portfolio made with [React](https://reactjs.org/)

## Quick Start

Download and run this application in development mode:

```
$ git clone https://github.com/sagerg/portfolio.git
$ cd portfolio
$ npm install
$ npm run serve
```

Run unit tests:

```
$ npm test
```

Create build directories `/dist` and `/docs`:

```
$ npm run build
```
